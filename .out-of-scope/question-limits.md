# Hard limits on the number of questions during grilling

The `/grill-me` skill does not enforce a maximum number of questions. Requests to add a configurable cap or hard ceiling are out of scope.

## Why this is out of scope

Grilling is intentionally open-ended. Some plans need three questions, some need fifty. A fixed cap would either cut off useful exploration on hard problems or feel arbitrary on easy ones. If a session feels too long, the right escape hatch is natural-language steering ("wrap up and summarise"), not a numeric limit.
