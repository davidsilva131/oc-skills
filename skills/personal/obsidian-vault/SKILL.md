# Obsidian Vault

Work with an Obsidian vault.

## Naming conventions

- Index notes for each topic area
- Title case for note names
- No folders — flat structure across the vault

## Linking

Use `[[wikilinks]]` to connect related notes. Each note should link to its parent index note and related concepts.

## Workflows

- **Search**: Find notes by filename or content
- **Create**: Create new notes with proper frontmatter and links
- **Find related**: Navigate via backlinks to discover related content
- **Index notes**: Central notes that list and link to all notes in a topic area
