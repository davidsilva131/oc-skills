# Writing Fragments

The **explore** phase of writing: mine raw fragments. No structure yet.

## What is a fragment

A sharp sentence, a claim, a vignette, a half-thought, a quote, a list, a leading word. The most valuable fragments are the ones that surprise you.

## Process

1. Run a grilling session that produces fragments
2. Append each fragment to a single markdown file
3. Separate fragments with horizontal rules

## Rules

- Append silently — don't edit or rearrange
- Re-read the full pile before each new fragment
- Never overwrite — fragments accumulate
- A session is done when the well runs dry
