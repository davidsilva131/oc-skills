# Loop Me

A stateful grilling session that produces workflow specs for recurring patterns in your life.

## Vocabulary

- **Trigger**: what starts the workflow
- **Checkpoint**: a step where you need to make a decision
- **Push right**: defer complexity to the next iteration
- **Brief**: a complete specification an implementer agent could build from

## Process

1. Identify a recurring pattern worth delegating
2. Define the trigger
3. Map the checkpoints
4. Write the brief
5. Repeat until the spec is implementable

## Definition of done

A workflow spec is done when an implementer agent could build it without asking questions.
